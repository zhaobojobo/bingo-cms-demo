# 編碼规范

## 防止SQL注入
自定義SQL查詢請使用PDO預處理語句及參數綁定方式，禁止將參數值直接拼接在SQL中

參考文檔：

- https://www.php.net/manual/zh/pdo.prepare.php
- https://www.php.net/manual/zh/pdostatement.execute.php

## 防止跨站脚本漏洞
- 請勿在代碼中直接引用 `$_GET`, `$_POST`，通過以下方式訪問
    - $_GET['參數名']: Helper:get(參數名，默認值)
    - $_POST['參數名']: Helper:get(參數名，默認值)
- 禁止在模板中嵌入`<script>代码...</script>`标记， 脚本代码应通过外部js文件引入
- 禁止在模板中嵌入`<style>代码...</style>`标记，样式代码应通过外部css文件引入
- 为 iframe添加 sandbox 属性 示例：`<iframe src="https://example.com" sandbox="allow-scripts"></iframe>`

## Html规范
- 为所有`<img>`标记添加`alt`属性
